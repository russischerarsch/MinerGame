# Coal Miner (Go)

## Описание
Coal Mine Tycoon учебный backend-проект на Go в жанре симулятора предприятия «Угольная шахта».

Игрок управляет предприятием: нанимает шахтёров, накапливает уголь, покупает оборудование и старается завершить игру за минимальное время.

Проект разработан для практики конкурентности в Go, работы с HTTP REST API и интерфейсами.

---

## Функциональность
- Пассивная генерация ресурсов  
- Найм шахтёров разных классов  
- Параллельная работа шахтёров через goroutines  
- Покупка оборудования  
- REST API для управления игрой  
- Graceful shutdown приложения  
- Сбор статистики предприятия  

---

## Стек технологий
- Go (Golang)  
- HTTP REST API  
- Goroutines & Channels  
- Context API  

### Основные концепции:
- Интерфейсы  
- Конкурентность  
- Контексты (`context.Context`)  
- Graceful shutdown  

---

## Архитектура

```go
type Miner interface {
	Run(ctx context.Context) <-chan Coal
	Info() MinerInfo
}
```

Каждый шахтёр реализует интерфейс `Miner` и работает как отдельная goroutine.

## Запуск приложения

1. Убедитесь, что установлен Go
