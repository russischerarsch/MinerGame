Проект: Экономическая игра «Угольная шахта» на Go.

Суть: Игрок управляет компанией, нанимает шахтёров (горутины), добывает уголь и покупает оборудование. Цель — купить всё оборудование за минимальное время.

Технологии:

Горутины, каналы, context (параллельная добыча угля)
sync.Mutex, атомики (конкурентность)
Интерфейс Miner (расширяемость)
REST API (Gin/gorilla/mux)
In-memory хранилище
Что умеет:

Найм шахтёров 3 классов
Покупка оборудования (3 вида)
HTTP API для управления
Graceful shutdown
Демонстрирует: конкурентность, интерфейсы, HTTP API, graceful shutdown.
